JavaScript アロー関数(Arrow Function)まとめ

基本構文

const 関数名 = (引数) => {
  処理内容
}

省略ルール

  • 引数が1つ:括弧 () を省略可能
  • 引数が0または2つ以上:括弧 () 必須
  • 中括弧 {} を省略した場合:暗黙の return が適用される
  • 中括弧あり:return を明示する必要あり

例:省略可能なパターン

引数1つ・中括弧省略(return省略)

const square = x => x * x;

引数なし・括弧必要

const greet = () => "Hello!";

引数2つ・括弧必要

const add = (a, b) => a + b;

複数行処理・return必要

const greet = name => {
  const msg = `こんにちは，${name}`;
  return msg;
};

Vue 3 Composition API での使用例

例1:setup内で定義する関数

import { ref } from 'vue';

export default {
  setup() {
    const count = ref(0);

    const increment = () => count.value++;

    return { count, increment };
  }
};

例2:イベントハンドラとして使用

<template>
  <button @click="increment">増やす</button>
  <p>{{ count }}</p>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(0);
const increment = () => count.value++;
</script>

例3:引数あり，暗黙のreturn

const double = x => x * 2;

例4:外部のthisを継承(Composition APIではthisは使わない)

const items = [1, 2, 3];
items.forEach(item => console.log(item));
